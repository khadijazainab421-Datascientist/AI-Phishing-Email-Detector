# AI Phishing Email Detector

## Project Overview

This project uses Natural Language Processing (NLP) and Machine Learning to detect phishing emails and classify them as phishing or legitimate messages.

## Dataset

* Total Emails: 82,486
* Features: Email Text
* Target Variable: Label
* Classes:

  * 0 = Legitimate Email
  * 1 = Phishing Email

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* TF-IDF Vectorization
* Logistic Regression

## Machine Learning Workflow

1. Data Loading
2. Data Exploration
3. Text Preprocessing
4. TF-IDF Feature Extraction
5. Train-Test Split
6. Model Training
7. Model Evaluation

## Results

* Accuracy: 98.18%
* Precision: 0.99
* Recall: 0.98
* F1 Score: 0.98

## Cybersecurity Application

The model helps identify phishing emails and can be used as an email security filtering system.
